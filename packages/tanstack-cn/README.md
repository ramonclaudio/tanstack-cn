# tanstack-cn

TanStack Start starter on Vite 8 + Oxc, Tailwind v4 + shadcn base-luma on Base UI. No Radix, no ESLint, no Prettier.

This package is a thin redirect. To scaffold a project, use [`create-tanstack-cn`](https://www.npmjs.com/package/create-tanstack-cn):

```bash
bunx create-tanstack-cn@latest my-app
# or
npm create tanstack-cn@latest my-app
```

Or clone the template directly:

```bash
bunx degit ramonclaudio/tanstack-cn/templates/default my-app
```

Running `bunx tanstack-cn` prints these same install instructions.
