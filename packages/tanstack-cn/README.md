# tanstack-cn

Redirect package. Running `npx tanstack-cn` (or `bunx`, `pnpm dlx`, `yarn dlx`) prints scaffold instructions and exits.

Use [`create-tanstack-cn`](https://www.npmjs.com/package/create-tanstack-cn):

```bash
npm create tanstack-cn@latest my-app
# or pnpm create, bunx create-tanstack-cn@latest, yarn create tanstack-cn
```

Or clone the template directly:

```bash
npx degit ramonclaudio/tanstack-cn/templates/default my-app
```

Repo: [`ramonclaudio/tanstack-cn`](https://github.com/ramonclaudio/tanstack-cn).
