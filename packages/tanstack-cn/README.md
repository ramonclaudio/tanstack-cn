# tanstack-cn

Thin redirect to [`create-tanstack-cn`](https://www.npmjs.com/package/create-tanstack-cn).

```bash
bunx create-tanstack-cn@latest my-app
# or
npm create tanstack-cn@latest my-app
# or
pnpm create tanstack-cn@latest my-app
```

Or clone the template directly:

```bash
bunx degit ramonclaudio/tanstack-cn/templates/default my-app
```

`bunx tanstack-cn` prints these same instructions and exits.

Repo: [ramonclaudio/tanstack-cn](https://github.com/ramonclaudio/tanstack-cn).
